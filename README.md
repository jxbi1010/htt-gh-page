# Heterogeneous Tactile Transformer — Project Page

Project page for **Heterogeneous Tactile Transformer (HTT)**, adapted from the
[Nerfies](https://github.com/nerfies/nerfies.github.io) template.

## Deploy on GitHub Pages

1. Create a repo (e.g. `HTT` or `htt.github.io`) and push the contents of this folder.
2. In **Settings → Pages**, set the source to the `main` branch, root (`/`).
3. The site will be served at `https://<user>.github.io/<repo>/`.

## Before publishing — fill in placeholders

- arXiv button URL: `https://arxiv.org/abs/XXXX.XXXXX`
- Code button URL: `https://github.com/jxbi1010/HTT`
- BibTeX `eprint` / `url` once the arXiv ID is assigned.
